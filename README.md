# 📘 Chains.ipynb

This notebook demonstrates how to use LangChain with Groq (Llama 3.1) to build simple LLM chains.

## What it includes

* Setting up Groq API
* Creating prompt templates
* Running an LLM chain
* Building a basic sequential chain

## Model

* llama-3.1-8b-instant

## Purpose

Learn how to connect prompts, models, and outputs to create simple AI workflows.
